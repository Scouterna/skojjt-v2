#codebase 
This project is called skojjt and is made to track attencance of scouts at scout meetings.
skojjt takes input in from the central member register for all scouts in sweden called scoutnet.
The persons using the system are leaders that use checkmarks  to register attendance for each person at a meeting.
The persons using need to log in since this is personal information in the system.
The output is reports of the attendance that can be sent to different institutions that can give the scout group money for each attendance mark.
The current system is running in Google App Engine with Python 3.11.
